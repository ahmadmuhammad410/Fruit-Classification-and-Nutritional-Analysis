# Fruit Classification and Nutritional Analysis System

An advanced computer vision project for automated fruit recognition and nutritional estimation using state-of-the-art deep learning architectures. This project combines transfer learning, few-shot learning, and metric learning techniques to deliver accurate fruit classification and calorie analysis across multiple AI models and pipelines.

---

## Key Features

- Multi-model deep learning framework using ConvNeXt, EfficientNetV2B0, Prototype Networks, and Siamese Networks
- Automated fruit classification across 30 fruit categories
- Nutritional and calorie estimation based on detected fruit type and weight metadata
- Transfer learning with ImageNet and CLIP pretrained backbones
- Few-shot learning support for low-data classification scenarios
- Metric learning with advanced Siamese Network architecture
- Production-ready inference pipeline with error handling and reporting
- Modular and scalable architecture for future deployment and extension

## Models Included

### 1. ConvNeXt-Base
- Transfer learning based fruit classification model
- Optimized for high-accuracy feature extraction

### 2. EfficientNetV2B0
- Lightweight and efficient inference model
- Supports calorie estimation pipeline

### 3. Prototype Network
- Few-shot learning implementation using CLIP embeddings
- Designed for low-data classification tasks

### 4. Siamese Network
- Research-grade metric learning model
- Uses triplet loss and semi-hard negative mining
